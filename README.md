# Telecom-X---An-lisis-de-Evasi-n-de-Clientes
En este proyecto realizamos la recopilación, procesamiento y analisis de los datos  con Python y sus principales bibliotecas para recolectar información valiosa para el desarrollo de estrategias, para una empresa de telecomunicaciones la cual estaba enfrentando una alta tasa de cancelaciones y perdidas de clientes.
